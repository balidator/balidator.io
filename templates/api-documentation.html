Blockchain address validator API documentation

Welcome to the documentation of 'Blockchain address validator'.

API Endpoint: https://balidator.io/api/

Parameters: /api/<coin>/<public_key>

Example: https://balidator.io/api/ethereum/0xea0258D0E745620e77B0A389e3A656EFdb7Cf821

JSON response:

{'blockchain': coin, 'valid_address': False}

We currently support the public key validation for:
  • Bitcoin
  • Ethereum
  • Ripple
  • Monero
You can always contribute to this open-source project. Just click the Github link in the footer and you will find our public repository.
